### Overview

This project is a golf scoring app. It will be used by competition organisers to calculate each layers scores and find the winner.
There are three pieces of information required. The golf course hole handicaps, the player details including handicap, and the players score on the day.
I have organised a few golf tournament and always had to make a complicated spreadsheet to calculate the scores. This will make my life a lot easier. 

### Features

User login.
There will be three types of users.
Players, who will be able to view all competitions that they have been registered for.
Organisers, who will be able to create and alter competitions.
Administrators, who will be able to amend golf course details.

Golf course search and creation.
Users will be able to search for specific golf courses and use those in their competition. If they don't exist then they can create a new course.

Competition creation.
Users will create the competition by adding the scoring type and all the player details.

Cometition scoring.
Users will enter the players scores and the app will calculate the winners based on set scores.
